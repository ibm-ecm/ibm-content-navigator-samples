IBM Content Navigator Toolkit Samples

This dynamic web project contains sample web pages demonstrating the use of the IBM Content Navigator toolkit. The samples are divided into three groups:

Model samples
These samples show the use of the ecm.model classes within the toolkit. They are mostly non-interactive in nature, running and completing by just viewing the page. Some have simple entry fields as well.
Dijit samples
These samples show the use of the ecm.widget (dijit) classes in a variety of configurations.
Layout sample
This sample is a full application, designed to show how a custom application can be built from the toolkit.
