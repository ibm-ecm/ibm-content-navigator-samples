Model Samples

These samples show the use of the ecm.model classes within the toolkit. They are mostly non-interactive in nature, running and completing by just viewing the page. Some have simple entry fields as well.

Login
Logs in a given repository
List ContentClasses
Displays ContentClasses of a given repository.
itemProperties.html
Displays the properties of a given item.
List SearchTemplates
Displays SearchTemplates of the given repository.
List TeamSpaces
Displays TeamSpaces of the given repository.
Search
Search documents with SearchTemplate.
Check Out
Check out the document.
Check In
Check in the document.
Download document
Download the document.
Add document
Add the document.
Viewer
Open the document with appropriate viewer.
Update Properties
Update properties of the document.
