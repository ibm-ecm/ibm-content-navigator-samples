IBM Content Navigator on Container for CMOD & CM8
=====================================================

This repository includes folders and resources to help you install the IBM Content Navigator for CMOD & CM8.
Installation of the capabilities is done with the operator.

The below sections provide information about the versions thats are deployed with the operator.
